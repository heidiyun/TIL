# Display
1. inline 
	* 기본 값
	* inline 말 그대로 요소가 띄어지지 않고 정렬된다.
	* <요소> <요소> <요소>
	* inline은 width와 height가 무시되고 contents의 크기를 따라간다.
	* span 처럼 적용됨
2. block 
	* 자신만의 구역이 생긴다고 생각하면 됨.
	* <요소>
	<요소>
	<요소>
	* p (단락) 처럼 적용됨
	* 요소 앞뒤로 줄 바꿈 적용.
3. contents
	* 겉 속성은 반영되지 않고 콘텐츠만 나온다. 
4. inline-block 
	* 정렬이 inline처럼 발생하지만, 첫 요소 앞과 마지막 요소 뒤에 줄 바꿈 발생
	* contents에 따라 width가 결정된다.
10. none
	* 요소가 화면에 표시되지 않는다.
